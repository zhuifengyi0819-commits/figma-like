# Phase 7: Complete Figma Feature Parity

## Goal
实现 Figma 所有核心设计功能，达到专业设计工具可用水平。

## Missing Features (Priority Order)

### P0 — Core Engine
1. **Rulers** — 水平和垂直标尺，带刻度
2. **Manual Guides** — 用户手动放置参考线，吸附到参考线
3. **Per-corner Radius** — 每个角独立圆角值
4. **Auto Layout Hugging/Frame Fill** — Auto Layout 的核心行为
5. **Multi-fill / Multi-stroke** — 每个 shape 支持多个填充和描边

### P0 — Components & Prototype
6. **Component States** — hover/active/pressed/disabled 状态渲染
7. **Prototype Scroll** — Frame 溢出时在原型播放模式可滚动
8. **whileDragging Trigger** — 拖拽时触发的原型交互

### P1 — Polish
9. **Component Library Panel** — 可复用组件浏览/搜索面板
10. **Layout Grid Overlay** — 编辑模式下显示 layout grid
11. **Pencil Tool** — freehand 画笔工具

## Deliverables
- 所有 P0 功能完整实现并可测试
- P1 功能核心实现
